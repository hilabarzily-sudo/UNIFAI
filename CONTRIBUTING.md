# 🤝 תרומה ל-UNIFAI

תודה על העניין לתרום לפרויקט UNIFAI! 

## 📋 איך לתרום?

### 1. Fork הפרויקט

לחץ על כפתור "Fork" בעמוד GitHub.

### 2. שכפל את ה-Fork שלך

```bash
git clone https://github.com/YOUR_USERNAME/UNIFAI.git
cd UNIFAI
```

### 3. צור ענף חדש

```bash
git checkout -b feature/amazing-feature
```

### 4. בצע שינויים

- כתוב קוד נקי וקריא
- הוסף הערות בעברית או אנגלית
- וודא שהעיצוב עקבי

### 5. Commit

```bash
git add .
git commit -m "הוסף: תכונה מדהימה"
```

### 6. Push

```bash
git push origin feature/amazing-feature
```

### 7. צור Pull Request

פתח Pull Request ב-GitHub עם תיאור מפורט.
