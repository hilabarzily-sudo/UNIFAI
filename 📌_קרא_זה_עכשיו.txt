🚀 איך להתחיל? (3 שלבים!)

  1️⃣  התקן תלויות (פעם אחת):
      
      npm install
      cd client && npm install && cd ..

  2️⃣  הוסף מפתח API (חינם!):
      
      → גש ל: https://makersuite.google.com/app/apikey
      → צור מפתח
      → פתח .env והדבק

  3️⃣  הרץ:
      
      Windows: לחץ על START_ALL.bat
      אחר: npm run dev:all

  4️⃣  פתח:
      
      http://localhost:5173
