## ⚡ Quick Start (3 Steps)

### 1. Install Dependencies (one time)

```bash
npm install
cd client && npm install && cd ..
```

### 2. Add API Key

Get a **free** Google Gemini key:
1. Go to: https://makersuite.google.com/app/apikey
2. Login with Google
3. Create API key
4. Copy it
5. Open `.env` and paste:
   ```
   GOOGLE_API_KEY=your_key_here
   ```

### 3. Run!

**Windows:** Double-click `START_ALL.bat`

**Mac/Linux:**
```bash
npm run dev:all
```

### 4. Open Browser

→ http://localhost:5173

**🎉 Done!**

## 🆘 Quick Help

### Issue: "Cannot find module"
→ Run: `npm install` in both root and client folders

### Issue: "Port already in use"
→ Change PORT in `.env` or kill the process

### Issue: "API key not configured"
→ Add key to `.env` and **restart server**

### Issue: Design looks broken
→ Ctrl+Shift+R (hard refresh) + check Console (F12)
