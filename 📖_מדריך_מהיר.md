## ⚡ התחלה מהירה

### צעד 1: התקן (פעם אחת - 3 דקות)

```bash
npm install
cd client && npm install && cd ..
```

### צעד 2: קבל מפתח חינם

**Google Gemini (חינם!):**

1. **לך ל**: https://makersuite.google.com/app/apikey
2. **התחבר** עם Gmail
3. **לחץ** "Create API key"
4. **העתק** את המפתח

### צעד 3: הדבק במחשב

1. **פתח** את הקובץ `.env`
2. **מצא** את השורה: `GOOGLE_API_KEY=`
3. **הדבק** את המפתח אחרי ה-`=`
4. **שמור**

דוגמה:
```
GOOGLE_API_KEY=AIzaSyABC123xyz...
```

### צעד 4: הרץ!

**Windows:**
```
לחץ פעמיים על: START_ALL.bat
```

**Mac/Linux:**
```bash
npm run dev:all
```

### צעד 5: פתח דפדפן

```
http://localhost:5173
```
