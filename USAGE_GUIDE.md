# 📖 UNIFAI - מדריך שימוש מפורט

מדריך מקיף לשימוש באפליקציית צ'אט בוט UNIFAI.

## 🎯 סקירה כללית

UNIFAI היא אפליקציית צ'אט בוט מתקדמת המשלבת מספר מודלי AI מובילים עם עיצוב glassmorphism מרהיב. האפליקציה מאפשרת לך לשוחח עם מודלי AI שונים ולהחליף ביניהם בקלות.

## 🚀 התחלת שימוש מהירה

### שלב 1: הכנה

1. **וודא שיש לך Node.js מותקן**
   ```bash
   node --version  # צריך להיות 18 ומעלה
   ```

2. **השג מפתחות API**
   - עבור OpenAI: https://platform.openai.com/api-keys
   - עבור Anthropic: https://console.anthropic.com/

### שלב 2: הגדרת המפתחות

1. העתק את קובץ `.env.example` ל-`.env`:
   ```bash
   cp .env.example .env
   ```

2. ערוך את `.env` והוסף את המפתחות שלך:
   ```env
   OPENAI_API_KEY=sk-proj-YOUR_KEY_HERE
   ANTHROPIC_API_KEY=sk-ant-YOUR_KEY_HERE
   ```

### שלב 3: הפעלה

**Windows:**
```bash
START.bat
```

**Mac/Linux:**
```bash
./START.sh
```

או באמצעות npm:
```bash
npm run dev
```

### שלב 4: פתח את הדפדפן

גש לכתובת: http://localhost:3000

## 💬 כיצד להשתמש בצ'אט

### שליחת הודעות

1. **הקלד את ההודעה שלך** בשדה הטקסט בתחתית המסך
2. **לחץ Enter** או על כפתור השליחה (📤)
3. **המתן לתשובה** - אייקון "Thinking..." יופיע בזמן שה-AI חושב

### טיפים לכתיבה:

- **Enter** - שולח את ההודעה
- **Shift + Enter** - שורה חדשה
- ניתן לכתוב הודעות ארוכות - השדה יתרחב אוטומטית

## 🤖 בחירת מודל AI

### כיצד להחליף מודל:

1. לחץ על כפתור המודל בכותרת (✨ שם המודל)
2. תיפתח חלונית עם רשימת המודלים הזמינים
3. בחר את המודל הרצוי
4. ההודעה הבאה תשלח למודל החדש

### מודלים זמינים:

- עלות: גבוהה

**GPT-4** 💎
- גרסה מקורית של GPT-4
- הקשר של 8K tokens
- איכות גבוהה מאוד
- עלות: גבוהה

**GPT-3.5 Turbo** ⚡
- מהיר וחסכוני

- מצוין לרוב המשימות
- עלות: נמוכה

#### Anthropic Models:

**Claude 3 Opus** 🎭
- המודל החזק ביותר של Anthropic
- מצוין לניתוח ועבודה עם טקסטים ארוכים
- הקשר של 200K tokens
- עלות: גבוהה

**Claude 3 Sonnet** 🎵
- איזון מושלם בין ביצועים למהירות
- הקשר של 200K tokens
- מומלץ לשימוש יומיומי
- עלות: בינונית

**Claude 3 Haiku** 🍃
- המהיר ביותר
- הקשר של 200K tokens
- מצוין למשימות פשוטות
- עלות: נמוכה

## 🎨 תכונות העיצוב

### Glassmorphism
האפליקציה משתמשת באפקט glassmorphism שכולל:
- **זכוכית מעורפלת** - רקע שקוף עם טשטוש
- **צללים עדינים** - לתחושת עומק
- **מסגרות עדינות** - גבולות שקופים
- **אנימציות חלקות** - מעברים ותנועות רכות

### פלטת צבעים
- גווני ירוק מונוכרומטיים
- השראה מטבע ושלווה
- צבעים מרגיעים לעיניים

### אנימציות
- הודעות מופיעות בצורה חלקה
- כפתורים מגיבים למגע העכבר
- רקע מונפש עם אורבים צבעוניים

## 🔧 התאמות אישיות

### שינוי פורט

ערוך את `.env`:
```env
PORT=8080  # במקום 5000
```

### שינוי טמפרטורה (יצירתיות)

ב-`client/src/services/api.js`, ניתן לשנות את פרמטר ה-temperature:
- **0.0** - דטרמיניסטי (תשובות זהות)
- **0.7** - ברירת מחדל (איזון)
- **1.0** - יצירתי (תשובות מגוונות)

### שינוי מודל ברירת מחדל

ערוך את `.env`:
```env
DEFAULT_MODEL=gpt-3.5-turbo  # או כל מודל אחר
```

## 🐛 פתרון בעיות נפוצות

### "API key not configured"
- וודא שהוספת מפתחות ב-`.env`
- הפעל מחדש את השרת

### "Model not available"
- המודל דורש מפתח API שלא הוגדר
- בחר מודל אחר או הוסף את המפתח הנדרש

### השרת לא עובד
```bash
# Windows
netstat -ano | findstr :5000
taskkill /PID <מספר_התהליך> /F

# Mac/Linux
lsof -ti:5000 | xargs kill
```

### הממשק לא טוען
1. נקה את ה-cache של הדפדפן (Ctrl+Shift+Delete)
2. בדוק את ה-console בדפדפן (F12)
3. וודא ש-backend רץ על פורט 5000
