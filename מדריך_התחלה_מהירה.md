# ⚡ UNIFAI - מדריך התחלה מהירה

התחל לעבוד עם צ'אט בוט AI מתקדם תוך 5 דקות!

## 📋 דרישות מקדימות

✅ Node.js גרסה 18 ומעלה  
✅ npm מותקן  
✅ מפתח API מ-OpenAI או Anthropic

## 🚀 התקנה בשלושה שלבים

### שלב 1: התקנת תלויות

```bash
npm run install-all
```

פקודה זו מתקינה את כל התלויות הנדרשות עבור ה-Backend וה-Frontend.

### שלב 2: הגדרת מפתחות API

```bash
# העתק את קובץ הדוגמה
cp .env.example .env
```

ערוך את הקובץ `.env` והוסף את המפתחות שלך:

```env
# הוסף לפחות מפתח API אחד:
OPENAI_API_KEY=sk-proj-המפתח_שלך_כאן
# או
ANTHROPIC_API_KEY=sk-ant-המפתח_שלך_כאן
```

**איפה מקבלים מפתחות API?**
- OpenAI: https://platform.openai.com/api-keys
- Anthropic: https://console.anthropic.com/

### שלב 3: הפעלת האפליקציה

```bash
npm run dev
```

**או השתמש בסקריפטים:**

Windows:
```bash
START.bat
```

Mac/Linux:
```bash
./START.sh
```

## 🌐 גישה לאפליקציה

פתח דפדפן וגש לכתובת:
```
http://localhost:3000
```

## 🐛 פתרון בעיות

### "Cannot find module"
```bash
npm run install-all
```

### "API key not configured"
1. וודא שקובץ `.env` קיים
2. בדוק שהמפתחות נכונים
3. הפעל מחדש את השרת

### הפורט תפוס
סגור את התהליך או שנה את הפורט ב-`.env`:

**Windows:**
```bash
netstat -ano | findstr :5000
taskkill /PID <מספר_תהליך> /F
```

**Mac/Linux:**
```bash
lsof -ti:5000 | xargs kill
```

### ה-Frontend לא נטען
1. בדוק שה-backend רץ: http://localhost:5000/api/health
2. נקה את ה-cache של הדפדפן
3. בדוק שגיאות ב-console (F12)
