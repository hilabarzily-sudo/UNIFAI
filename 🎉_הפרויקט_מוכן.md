### צעד 1: התקן (פעם אחת - 3 דקות)

```bash

cd client && npm install && cd ..
```

### צעד 2: הוסף מפתח (חינם! - 2 דקות)

```
1. לך ל: https://makersuite.google.com/app/apikey
2. התחבר עם Gmail
3. Create API key
4. העתק
5. פתח .env
6. הדבק ב-GOOGLE_API_KEY=
7. שמור
```

