```

2. Login with Gmail
3. Create API key
4. Copy key (starts with AIza...)
5. Paste in .env file:
   GOOGLE_API_KEY=AIza...
6. Save
7. Restart server!
```

### OpenAI (משולם)

```
1. גש ל: https://platform.openai.com/api-keys
2. Create account (needs credit card)
3. Create secret key
4. Copy key (starts with sk-proj- or sk-)
5. Paste in .env:
   OPENAI_API_KEY=sk-...
```

### Anthropic (משולם)

```
1. גש ל: https://console.anthropic.com/
2. Create account
3. Get API key
4. Copy key (starts with sk-ant-)
5. Paste in .env:
   ANTHROPIC_API_KEY=sk-ant-...
```

## 🆘 עזרה מהירה

### בעיה: "Cannot find module"

```bash
rm -rf node_modules package-lock.json
npm install
cd client
rm -rf node_modules package-lock.json
npm install
cd ..
```

### בעיה: "Port already in use"

**Windows:**
```cmd
netstat -ano | findstr :3001
taskkill /PID <PID> /F
```

**Mac/Linux:**
```bash
lsof -ti:3001 | xargs kill -9
```

### בעיה: "API key not configured"

1. פתח `.env`
2. וודא שהמפתח נכתב נכון
3. אין רווחים
4. **הפעל מחדש שרת!**

### בעיה: עיצוב שבור

1. `Ctrl + Shift + R` (hard refresh)
2. F12 → Console (בדוק שגיאות)
3. בדוק שהפונט נטען (Network tab)
