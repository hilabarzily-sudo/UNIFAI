## ⚡ התחלה תוך 5 דקות

### 1️⃣ התקן תלויות (פעם אחת)

```bash
npm install
cd client && npm install && cd ..
```

⏱️ **זה ייקח 2-3 דקות**

### 2️⃣ קבל מפתח API חינם

<div align="center">

**👉 לך לכאן: https://makersuite.google.com/app/apikey 👈**

</div>

1. התחבר עם Gmail
2. לחץ "Create API key"
3. העתק את המפתח

### 3️⃣ הוסף את המפתח

פתח את הקובץ **`.env`** ומצא:

```env
GOOGLE_API_KEY=
```

הדבק את המפתח אחרי ה-`=`:

```env
GOOGLE_API_KEY=AIzaSyABC123...
```

שמור את הקובץ!

### 4️⃣ הרץ!

<div align="center">

**Windows:**  
לחץ פעמיים על `START_ALL.bat`

**Mac/Linux:**  
```bash
npm run dev:all
```

</div>

### 5️⃣ פתח בדפדפן

<div align="center">

## 🌐 http://localhost:5173

## ❓ שאלות נפוצות

### האפליקציה לא עולה?

```bash
# נקה והתקן מחדש
rm -rf node_modules
npm install
cd client
rm -rf node_modules
npm install
```

### מקבל "API key not configured"?

1. פתח `.env`
2. בדוק שהמפתח נכתב נכון
3. **הפעל מחדש את השרת!**

### העיצוב לא נראה טוב?

1. `Ctrl + Shift + R` (רענון מלא)
2. `F12` → Console (בדוק שגיאות)
3. וודא שהפונט Heebo נטען
