## ⏱️ התחלה מהירה (3 דקות)

### שלב 1: פתח Terminal/CMD

```bash
cd /workspace
```

### שלב 2: התקן תלויות (פעם אחת!)

```bash
npm install
cd client
npm install
cd ..
```

⏳ **זה ייקח 2-3 דקות בפעם הראשונה**

### שלב 3: הוסף מפתח API חינם!

1. **גש לכאן**: https://makersuite.google.com/app/apikey
2. **התחבר** עם Gmail
3. **לחץ** "Create API key"
4. **העתק** את המפתח
5. **פתח** את הקובץ `.env`
6. **הדבק**:
   ```
   GOOGLE_API_KEY=המפתח_שהעתקת
   ```
7. **שמור** את הקובץ

### שלב 4: הרץ!

**Windows:**
```
לחץ פעמיים על START_ALL.bat
```

**Mac/Linux:**
```bash
npm run dev:all
```

### שלב 5: פתח דפדפן

```
http://localhost:5173
```

## 🎯 מה יש כאן?

### Backend (שרת)

```
server/
├── index.js              ← נקודת כניסה
├── routes/
│   ├── chat.js          ← API של צ'אט
│   └── models.js        ← רשימת מודלים
└── services/
    └── ChatService.js   ← מדבר עם AI
```

**מה זה עושה?**
- מקבל הודעות ממך
- שולח ל-AI (GPT/Claude/Gemini)
- מחזיר תשובות

### Frontend (ממשק)

```
client/src/
├── components/          ← 9 קומפוננטות!
│   ├── Header           ← כותרת + לוגו
│   ├── FloatingBalloons ← בלונים צפים
│   ├── ModelSelector    ← בחירת מודל
│   ├── ChatContainer    ← מכיל צ'אט
│   ├── MessageList      ← רשימת הודעות
│   ├── Message          ← הודעה בודדת
│   ├── MessageInput     ← תיבת קלט
│   └── TypingIndicator  ← נקודות מהבהבות
├── App.jsx              ← ראשי
└── index.css            ← סגנונות
```

**מה זה עושה?**
- מציג ממשק יפהפה
- מאפשר לדבר עם AI
- מונפש וחלק

## ⚠️ שגיאות נפוצות ופתרונות

### "Cannot find module"

```bash
# פתרון:
rm -rf node_modules
npm install
cd client
rm -rf node_modules
npm install
```

### "Port 3001 already in use"

```
פתרון 1: שנה PORT ב-.env
פתרון 2: הרוג תהליך ישן
```

### "API key not configured"

```
1. בדוק ש-.env קיים
2. בדוק שהמפתח נכון
3. הפעל מחדש את השרת!
```

### העיצוב לא נראה טוב

```
1. Ctrl+Shift+R (רענון מלא)
2. F12 → Console (בדוק שגיאות)
3. בדוק שהפונט נטען
```
