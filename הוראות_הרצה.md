# 🎈 UNIFAI - הוראות הרצה בעברית

## 📝 מה קיבלת?

יצרתי עבורך אפליקציית צ'אט בוט מתקדמת עם:

✅ **Backend חכם** - שרת Node.js שמדבר עם 3 ספקי AI  
✅ **Frontend מהמם** - React עם עיצוב Glassmorphism  
✅ **עיצוב ייחודי** - בלונים צפים וצבעים פסטליים  
✅ **תמיכה מלאה בעברית** - RTL וממשק בעברית  
✅ **מודלים מרובים** - GPT-4, Claude 3, Gemini Pro  

## 🚀 איך להריץ? (3 דקות!)

### שלב 1: פתח את הפרויקט

```bash
cd /workspace
```

### שלב 2: התקן תלויות (פעם אחת בלבד!)

```bash
# התקן תלויות השרת
npm install

# התקן תלויות הקליינט
cd client
npm install
cd ..
```

**⏱️ זה ייקח כ-2-3 דקות בפעם הראשונה**

### שלב 3: הוסף מפתח API

פתח את קובץ `.env` והוסף **לפחות מפתח אחד**:

```env
# OpenAI (מומלץ להתחיל איתו)
OPENAI_API_KEY=sk-proj-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx

# או Anthropic
ANTHROPIC_API_KEY=sk-ant-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx

# או Google (בחינם!)
GOOGLE_API_KEY=AIzaxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```

**איפה מקבלים מפתחות?**
- OpenAI: https://platform.openai.com/api-keys
- Anthropic: https://console.anthropic.com/
- Google: https://makersuite.google.com/app/apikey

### שלב 4: הרץ!

**אופציה 1 - Windows (קל ביותר):**
לחץ פעמיים על הקובץ:
```
START_ALL.bat
```

**אופציה 2 - Command Line:**
```bash
npm run dev:all
```

**אופציה 3 - בנפרד (2 חלונות terminal):**

חלון 1 (Backend):
```bash
npm run dev
```

חלון 2 (Frontend):
```bash
npm run dev:frontend
```

### שלב 5: פתח בדפדפן

גש ל: **http://localhost:5173**

🎉 **זהו! האפליקציה רצה!**

## ❓ שאלות נפוצות

### ש: האפליקציה לא עולה?

**בדוק:**
1. האם Node.js מותקן? הרץ: `node --version` (צריך v18+)
2. האם רצת `npm install`?
3. בדוק שאין שגיאות אדומות ב-terminal

### ש: מקבלת "Failed to fetch models"?

**פתרון:**
1. וודא שהשרת רץ (הרץ `npm run dev` בחלון נפרד)
2. בדוק שהשרת על פורט 3001
3. רענן את הדפדפן (Ctrl+R)

### ש: "API key not configured"?

**פתרון:**
1. וודא שקובץ `.env` קיים בתיקייה הראשית
2. בדוק שהמפתח נכתב נכון (ללא רווחים!)
3. **הפעל מחדש את השרת** אחרי שינוי `.env`

### ש: העיצוב לא נראה טוב?

**פתרון:**
1. רענן את הדפדפן עם Ctrl+Shift+R (ניקוי cache)
2. בדוק שאין שגיאות ב-Console (F12)
3. וודא שהפונט Heebo נטען (בדוק ב-Network tab)

### ש: איך משנים את הצבעים?

עדכן ב-`client/src/index.css` את המשתנים:

```css
:root {
  --primary-pink: #הצבע_שלך;
  --primary-blue: #הצבע_שלך;
  /* ... */
}
```

## 🔑 איפה מקבלים מפתחות API?

### OpenAI (מומלץ!)

1. גש ל: https://platform.openai.com/signup
2. צור חשבון (דורש כרטיס אשראי)
3. לך ל-API Keys: https://platform.openai.com/api-keys
4. לחץ "Create new secret key"
5. **העתק את המפתח** (מתחיל ב-`sk-proj-` או `sk-`)
6. הדבק ב-`.env` ליד `OPENAI_API_KEY=`

**💰 עלות**: 
- GPT-3.5: זול מאוד (~$0.002 לשאלה)
- GPT-4: יקר יותר (~$0.03 לשאלה)

**💡 טיפ**: תוכל לקבל $5 קרדיט חינם!

### Google Gemini (חינם!)

1. גש ל: https://makersuite.google.com/app/apikey
2. התחבר עם Google
3. לחץ "Get API key" → "Create API key"
4. **העתק את המפתח**
5. הדבק ב-`.env` ליד `GOOGLE_API_KEY=`

**💰 עלות**: **חינם!** (עם הגבלות סבירות)

### Anthropic Claude

1. גש ל: https://console.anthropic.com/
2. צור חשבון
3. Navigate to API Keys
4. צור מפתח חדש
5. **העתק** והדבק ב-`.env`
