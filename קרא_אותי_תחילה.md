## ⚡ התחלה מהירה (60 שניות!)

### 1. פתח Terminal/CMD

```bash
cd /workspace
```

### 2. התקן (פעם אחת - 2-3 דקות)

```bash
npm install
cd client && npm install && cd ..
```

### 3. הוסף מפתח AI

פתח את הקובץ `.env` והוסף מפתח (מומלץ Google - חינם!):

```env
GOOGLE_API_KEY=המפתח_שלך_כאן
```

**איפה מקבלים?** https://makersuite.google.com/app/apikey

### 4. הרץ!

**Windows:** 
```
לחץ פעמיים על START_ALL.bat
```

**Mac/Linux:**
```bash
npm run dev:all
```

### 5. פתח בדפדפן

```
http://localhost:5173
```

**🎉 זהו! תהני!**

### בעיה: "Cannot find module"

```bash
npm install
cd client && npm install && cd ..
```

### בעיה: "Port already in use"

שנה ב-`.env`:
```env
PORT=3002
```

1. פתח `.env`
2. הוסף מפתח
